接口地址：/api
备用接口地址：/api2
返回格式：JSON
请求方式：GET/POST

请求参数说明：

名称 必填 类型 说明
  cityname 是 string 请输入城市的拼音，例如: nanjing

返回参数说明：

名称 类型 说明
  image string 返回图片URL
  word string 返回一言
  voice string 返回音频URL
  date string 返回日期
  weather list 返回明日天气(明天的日期、天气情况、最高温度、最低温度、风级)

服务级错误码参照(error_code)：

  错误码 说明
  20011 无效的请求

系统级错误码参照：

  错误码 说明
  10011 无效的请求

错误码格式说明（示例：20011）：

  2 001 1
  服务级错误（1为系统级错误） 服务模块代码(即数据ID) 具体错误代码

Python演示：

import requests

if __name__=="__main__":
    url = 'http://lean-api.wxiou.cn/api'
    #url = 'http://lean-api.wxiou.cn/api2'
    
    datas ={
        'cityname':"nanjing"
    }
    respose = requests.get(url=url,data=datas)
    # status_code 为返回的状态码
    print(respose.status_code)
    # text为返回的数据
    print(respose.text)
